# RSL-DCE-control-program
This repository contains a MATLAB-based command-line control program for the Digital Channel Emulator (DCE). The program reads a satellite pass profile generated by MATLAB, parses the RF link parameters, and sends real-time commands to the programmable attenuator and software-defined radio (SDR). The satellite pass profile contains time-varying RF link characteristics that emulate the changing conditions experienced during a satellite pass.

# Reqired Toolboxes
MATLAB R2023a or newer
Instrument Control Toolbox
Satellite Communications Toolbox
Phased Array System Toolbox

# Workflow
1. Generate satellite pass profiles using runPassPrediction.m.
2. The data will be exported as CSV files contained in the CANX2 Data directory.
3. Run the hardwareLink.m and select the desired CSV file.
4. The program reads the attenuation profile and sends real-time commands to the programmable attenuator and software defined radio, reproducing the RF link conditions expected during the satellite pass.

# How to Use

1) runPassPrediction.m (Generate Satellite Pass Profiles)
From the MATLAB Command Window:

runPassPrediction();

runPassPrediction( ...
    TLEFile="ISS.tle", ...
    SatName="ISS", ...
    GSLat=49.2606, ...
    GSLon=-123.2460);

From an OS terminal (without opening the MATLAB desktop):

matlab -batch "runPassPrediction"

matlab -batch "runPassPrediction(TLEFile='CANX-2.tle', SampleTime=10)"

2) hardwareLink.m (Generate the RF Link in DCE)

Run file from MATLAB. When prompted, select one of the generated CSV files. The program will replay the attenuation profile in real time by sending commands to the programmable attenuator and SDR.

# Output

Each generated CSV file contains the time-varying RF link parameters for a single satellite pass, including:

Range (m)
Azimuth (deg)
Elevation (deg)
Path Loss (dB)
Propagation Delay (s)
Doppler Shift (Hz)
Relative Velocity (m/s)

These parameters are used by the hardware control program to emulate realistic satellite communication link conditions in real time.
